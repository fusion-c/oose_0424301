##專題簡介

#Pwoer Sport 體育用品網站.
##Youtube視頻專題簡介 <https://youtu.be/b1oXZ6G64rk>

我們小組打算做一個綫上販賣體育用品的B2C網站。這個網站可以讓使用者注冊賬戶，在使用者登入後可選購商品，
然後放入購物車，使用者可以刪除或更改購物的品項目。當使用者選購完商品後可把購物車中的商品一起下單結賬
，并且在下單的一個小時内可以取消訂單。
